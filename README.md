# Electricity Bill Estimator

An Android application for estimating monthly electricity bills 
based on TNB (Tenaga Nasional Berhad) tariff rates.

## Features
- Calculate electricity bill using block tariff system
- Apply rebate percentage (0% - 5%)
- Save, view, edit and delete bill records
- Bill history with detailed view

## Tariff Rates
| Block | Rate |
|-------|------|
| First 200 kWh | 21.8 sen/kWh |
| Next 100 kWh | 33.4 sen/kWh |
| Next 300 kWh | 51.6 sen/kWh |
| Next 300 kWh | 54.6 sen/kWh |

## Built With
- Android Studio
- Java
- SQLite Database
